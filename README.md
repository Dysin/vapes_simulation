## 一、仿真目标背景

电子烟气道的仿真主要通过 **单相流（空气）** 或 **双相流（气液两相）** 模型，分析流体在气道内的流动特性，从而反映出使用者的**吸阻**、**流感**、**烟雾输运**和**冷凝特性**。

在单相流阶段，我们主要聚焦以下几类参数：

> 1️⃣ 流动动力学参数（速度、压力、涡量等）
>
> 2️⃣ 能量与传热参数（温度分布）
> 
> 3️⃣ 流场结构特征（分离、再附着、湍流强度等）
> 
> 4️⃣ 体感相关二次指标（吸阻、均匀性、雾量传输能力）

## 🧮 五、后续可扩展指标（建立口感数据库时）

1. **吸阻系数化指标**
   [
   C_{res} = \frac{ΔP_{total}}{0.5ρU_{in}^2}
   ]
   用于不同设计的无量纲比较。

2. **均匀性系数**
   [
   K_{u} = 1 - \frac{\sigma(U)}{\bar{U}}
   ]
   衡量出口气流均匀度（越高越顺畅）。

3. **综合口感指数（可用于数据库）**
   [
   I_{taste} = w_1 \cdot C_{res} + w_2 \cdot (1-K_u) + w_3 \cdot Tu + w_4 \cdot T_{out}
   ]
   （可拟合主观口感测试数据）

# 气动噪声

## 一、气动噪声产生的本质与关键参数

电子烟气道的气动噪声（Aeroacoustic noise）主要源于：

* 气流通过狭窄孔、弯道、接缝等处的**非定常涡结构与剪切层相互作用**；
* 局部压力和速度波动（尤其是**涡崩塌和再附着**）；
* 与固体壁面的相互作用（壁面压力脉动）；
* 射流、收缩、喉部附近的**流速梯度变化**。

因此 CFD 中需要考虑的关键物理量包括：

| 类别             | 代表参数                                                              | 含义 / 与噪声关系                  |
| -------------- | ----------------------------------------------------------------- | --------------------------- |
| **速度场**        | ( u', v', w' )、( \text{mag}(\mathbf{u}) )                         | 局部速度波动幅度越大 → 噪声越强           |
| **压力场**        | ( p', \frac{dp}{dt} )、RMS(p)                                      | 声源直接来自压力脉动；RMS 表示声强         |
| **涡量 / 涡核特征**  | ( \omega = \nabla \times \mathbf{u} )、Q 值、λ₂ 值                    | 涡结构的破碎与脱落对应宽频噪声源            |
| **湍流参数**       | ( k )（湍动能）、( \varepsilon )、TI（湍流强度）                               | 湍流能越高，随机压力脉动越强              |
| **速度梯度 / 剪切率** | ( \partial u / \partial y ) 等                                     | 强剪切层是高频噪声的重要来源              |
| **Re数 & Ma数**  | ( \text{Re} = \frac{\rho U D}{\mu} ), ( \text{Ma} = \frac{U}{a} ) | Re高 → 涡更复杂；Ma>0.3时压缩性效应开始显著 |
| **壁面压力波动**     | ( p'_{wall}(t) )                                                  | 直接决定用户听到的“气流声”与“喉感”         |

---

## 二、在 RANS 与 DDES 模型中的描述差异

| 对比项    | **RANS (Reynolds-Averaged Navier-Stokes)** | **DDES (Delayed Detached Eddy Simulation)** |
| ------ | ------------------------------------------ | ------------------------------------------- |
| 主要目标   | 求时间平均流场                                    | 同时解析平均流 + 大尺度非定常涡结构                         |
| 能否捕捉声源 | ❌ 仅平均量（压力、速度） → 无真实声波信息                    | ✅ 可解析真实压力脉动与涡脱落过程                           |
| 声学适用性  | 可用作统计估算，如：基于平均湍动能推声源强度（经验公式）               | 可用于气动声直接模拟（Hybrid LES 精度）                   |
| 声源建模   | 使用基于RANS场的经验声源模型（如 Proudman 公式、Curle 源项）   | 直接在非定常流场中提取声压信号                             |
| 计算代价   | 低                                          | 中高（视网格与时间步）                                 |

👉 因此：

* **RANS → 只能估算噪声趋势**；
* **DDES（或LES） → 可获得实际频谱、声源分布、时间波形。**

---

## 📈 三、判断高频与低频噪声的方式（频谱角度）

1. **压力信号时域采样**

   * 在气道关键位置（进气口、喉部、烟嘴处壁面）放置探针；
   * 输出瞬时压力 ( p(t) )。

2. **做傅里叶变换（FFT）或功率谱密度（PSD）**

   * 转换得到声压频谱 ( P(f) )。

3. **频段划分（经验）**

   * **低频噪声**（< 1 kHz）：
     来自大尺度涡团、整体流场波动；声感为“低沉、呼呼声”。
   * **中频噪声**（1–5 kHz）：
     与气道收缩、扩张的流体分离振荡有关。
   * **高频噪声**（> 5 kHz）：
     小尺度湍流、剪切层破碎；声感为“尖锐嘶嘶声”。

4. **Star-CCM+ 中常用流程：**

   * 运行 **Transient Simulation (DDES/LES)**；
   * 用 **Surface Monitor → Pressure** 记录时间序列；
   * 导出 `.csv`；

## 🧠 四、总结与工程判断要点

| 项目       | 主要关注点                 | 工程意义              |
| -------- | --------------------- | ----------------- |
| **RANS** | 平均压力分布、湍动能分布、局部速度梯度   | 识别可能的噪声源区（喉部、收缩区） |
| **DDES** | 瞬时压力波动、涡结构脱落频率、声压谱    | 定量分析声源频率与声强       |
| **高频噪声** | 小尺度湍流（剪切层、喷口、壁面交互）    | 嘶嘶声，尖锐感           |
| **低频噪声** | 大尺度涡、流动不稳定、周期性分离      | 低沉呼吸声或“呼呼声”       |
| **控制策略** | 优化气道形状、平滑过渡、减少急弯、调节孔径 | 降噪同时改善口感平稳度       |


# Q涡判断

# 1) Q 的物理与数学定义（要精确）

令速度梯度张量 ( \nabla \mathbf{u} ) 分解为对称的应变率张量 ( \mathbf{S} ) 与反对称的旋转率张量 ( \mathbf{\Omega} )：
[
\mathbf{S} = \tfrac{1}{2}\left(\nabla\mathbf{u} + (\nabla\mathbf{u})^T\right),\qquad
\mathbf{\Omega} = \tfrac{1}{2}\left(\nabla\mathbf{u} - (\nabla\mathbf{u})^T\right)
]
Q-criterion 定义为：
[
Q = \tfrac{1}{2}\left(|\mathbf{\Omega}|^2 - |\mathbf{S}|^2\right)
]
当 (Q>0) 时，旋转（离心义）主导，应被视为涡核／旋涡区；当 (Q<0) 时，应变（拉伸/压缩）主导。

（另外可比较的判识量：(\lambda_2)、swirl-strength、vorticity magnitude，但 Q 对区分“旋转主导 vs 应变主导”很直观。）

---

# 2) 如何在 CFD 后处理里计算与可视化（工具/流程）

* 工具：ParaView、Tecplot、StarCCM+ 的 field function、或自写 Python 后处理（numpy）都可。
* 输入：瞬时速度场（若关注瞬态涡），或时间序列用于时间平均/统计（若关注长期特征）。
* 计算步骤：

  1. 计算速度梯度张量 (\partial u_i/\partial x_j)（中心差/高阶差分）；
  2. 求 (\mathbf{S})、(\mathbf{\Omega})、并得到 (Q) 场；
  3. 生成 Q>0 的等值面（isosurface）或平面等高线；也可绘制 Q 的瞬时剖面和时间平均 (\overline{Q})、RMS(Q)。
* 可视化建议：

  * 在关键截面（咪头附近、雾化区入口/出口、弯道）的 Q 等值面 + 流线共存；
  * 把 Q 等值面着色（例如按 Q 值或按局部压力/湍动能）；
  * 对瞬态 LES/URANS 数据，生成 Q 动画（帧）以观察涡脱落频率。

---

# 3) 量化涡特征（指标化，便于比较不同设计）

* **peak (Q)**：局部最大值（反映最强旋涡强度）。
* **体积分布 (V_{Q>!Q_{th}})**：阈值 (Q_{th}) 之上体积（反映占比/严重程度）。建议用**分位法**设阈值（例如取场中 95 百分位），比固定绝对阈值更稳健。
* **涡核计数与连通性**：连通体个数（多少个独立涡核）。
* **涡轴长度 / 位置**：每个涡核的几何尺寸与中心位置（相对于嘴/雾化芯）。
* **循环强度 Γ（circulation）**：在涡核周围积分切向速度，衡量对气动力的实际影响。
* **Q-weighted centerline / 重心**：表示涡活动集中区域。
* **时间谱/频率**：对某点 Q(t) 做 FFT，得到主频（可用于估算可能的啸叫/噪声 Strouhal number）。
* **归一化 Q**：把 Q 归一化成无量纲 (Q^* = Q \cdot (D/U)^2) 或 (Q/U^2/D^2)，便于不同工况之间比较。

---

# 4) 如何用 Q 判别气道特征与问题根源（诊断表）

* **分离/回流区**：Q 的高值连同压力回升或负流速区（逆流）出现在锐角、突变截面或急扩处 → 说明分离导致能量损失与噪声。
* **剪切层/喷射射流**：在窄口/喉部出口处出现强 Q 且伴随高湍动能 → 强剪切层，有利混合但可能引起喉感刺激与雾滴不均。
* **涡脱落（频繁、规则）**：时间序列出现明显主频 → 可能产生低频脉动噪声或频率相关的喉感。
* **“死腔”/滞留区**：Q≈0 但速度非常低、并伴随湍动能低且停滞 → 导致液体滞留、沉积或雾化不均匀。
* **高 Q 与高压降共现**：表示旋涡与分离导致显著局部能量损失（目标优先处理）。

---

# 5) 针对不同问题的具体优化策略（从局部到系统性）

（每条都写明诊断信号 → 对应改法）

1. 分离/强涡（诊断：Q 高、逆流、压力回升）

   * 改法：增加圆角/倒角，延长收缩/扩张过渡长度（锥长≥直径的 1–3 倍做初试），降低扩散角度（扩散器<7°–10° 建议值）；避免急剧截面突变。
2. 喉部/喷射剪切层过强（诊断：喉口 Q 高、湍动能高）

   * 改法：略增喉径或减小局部速度；或在喉后设置渐开导流片/微扰片以控制剪切层发展（注意可能增压降）。
3. 规则涡脱落带来噪声（诊断：Q(t) 有强主频）

   * 改法：打乱规律结构（如微孔分流、非对称导流、微扰装置）以破坏相干涡列；或者改变局部尺度使 Strouhal 移出敏感频段。
4. 死腔/滞留（诊断：速度低、Q≈0）

   * 改法：缩小死腔体积，增设通气导流或轻微倾斜/斜面使液体不易滞留。
5. 局部尖锐突起与装配缝隙（诊断：Q 高、涡在接缝周围）

   * 改法：优化装配、遮挡缝隙处、或把缝隙移到非流动核心区域。
6. 要求保留湍流以提升混合但降低喉感：做 **局部“可控湍流”** —— 把高湍流保留在雾化区入口（利于雾化），在出气口前通过扩压/逐步减速来降低湍流强度（平顺的恢复区）。

---

# 6) 优化流程（工程实践）

1. **识别 & 定位**：用 Q、压力、速度剖面联合定位问题段（instantaneous + time-avg）。
2. **小改测试**：在 CAD 做 1–3 个小变体（fillet、锥长、喉径变动），用 RANS 快速筛（steady 或 URANS）。
3. **细化验证**：对最优候选做 LES/WALE 或 DES 的短时解析，观察瞬态涡脱落/频谱。
4. **多目标权衡**：把 Δp（压降）、Q 指标（体积或 peak）、湍流强度、以及口感/吸阻指标放到 DOE 或优化器里做 Pareto。
5. **生产验证**：原型制造、流量与压降台架测试、感官试验（主观口感验证）与 CFD 对照。

---

# 7) CFD/数值设置建议（为得到可靠 Q 场）

* **网格与分辨率**：若主要关注涡结构，RANS 常常低估瞬态小尺度涡；建议 URANS/LES/WALE 或 DES。LES 对小尺度涡敏感，但成本高；DES/URANS 可做折中。
* **Near-wall**：若局部分离很关键，保证足够近壁解析（y+小于 1 或使用能解析近壁的模式）；RANS 用壁函数则遗漏部分细尺度涡。
* **时间步**：Δt 需能解析主要涡脱落频率（CFL < 1 或至少满足频谱要求）。
* **后处理采样**：对时间场做长时间统计（至少若干个主要涡周期）以得到稳定的平均 Q 与频谱。

# 局部压力过大问题

非常好的问题，这个点非常关键——**局部压力差过大**是电子烟气道设计中常见但又容易被忽视的问题之一，它直接影响**口感、气流稳定性以及冷凝液和积液风险**。下面我从**流体力学角度 + 用户体验角度 + 工程优化角度**详细说明。

---

### 🚬 一、物理机制：局部压差大的原因

局部压差（pressure gradient）通常出现在以下位置：

* **截面积骤变**（例如气道收缩或突然扩张）；
* **锐角拐弯**；
* **结构连接缝隙**（如烟嘴与咪头、主气道连接处）；
* **气道表面粗糙或存在突起**；
* **二次流或回流区**（通常在涡结构强烈区域）。

这些都会导致局部流速急剧变化、能量损失上升，表现为局部压差显著。

---

### 💨 二、结果与现象：局部压差大的后果

| 影响类别             | 具体表现         | 机理说明                   |
| ---------------- | ------------ | ---------------------- |
| **1. 抽吸阻力增大**    | 用户感觉“难吸”“闷”  | 局部高压损导致系统总压降上升         |
| **2. 流场不均匀**     | 出气速度不稳，烟雾流不均 | 局部流速过高或过低，导致速度分布非对称    |
| **3. 涡流增强，噪声增加** | 吸气时有气流“啸声”   | 压差驱动强涡结构（Q值高区）导致气动噪声   |
| **4. 冷凝液、积液风险**  | 烟嘴处积液或倒流     | 局部低压引发二次回流，烟雾冷凝沉积      |
| **5. 味道与口感劣化**   | 味道不稳定、口腔刺激不均 | 局部湍流与温度分布不均导致气溶胶颗粒输运异常 |

---

### 🧭 三、优化思路与手段

#### ✅ 1. **几何优化**

* **平滑过渡**：避免突变截面，使用圆角或贝塞尔曲线过渡；
* **减小弯曲半径变化率**：使气流路径更加柔顺；
* **优化截面形状**：从圆→椭圆→矩形过渡时，应保持流体体积连续；
* **避免急剧扩张**：扩张角度<15° 可显著减少分离流。

#### ✅ 2. **流场分析优化**

* 在 **StarCCM+ 中查看 Q-criterion 等涡结构云图**：

  * **高 Q 值区通常对应强局部剪切或分离流区**；
  * 这些区域常伴随**局部低压区**；
  * 若 Q 值与静压梯度区域重叠，则说明该位置存在能量损失热点。
* 优化目标：

  * **让 Q 值分布更连续，避免局部高 Q 峰值团块；**
  * **保证流线在该处连续顺滑，分离区消失或缩小。**

#### ✅ 3. **多物理场考虑**

* **气动 + 热场 + 传质耦合**：压差大处往往是冷凝和温度梯度突变区；
* 可联合考虑**流速、压强、温度、饱和度**等，评估冷凝风险。

#### ✅ 4. **CFD后处理指标**

* ( \Delta P = P_{\text{local,max}} - P_{\text{local,min}} )
* ( \Phi = \frac{\Delta P}{P_{\text{inlet}} - P_{\text{outlet}}} )
  若局部压差贡献比 (\Phi > 0.3)，说明该位置为关键能量损失点，应重点优化。

---

### 🧠 四、总结建议

| 问题        | 优化手段            |
| --------- | --------------- |
| 局部压差高     | 平滑过渡、优化截面、减少收缩比 |
| 流场不稳、Q值过高 | 缓解涡结构，控制边界层分离   |
| 噪声大       | 降低局部速度梯度、避免强剪切  |
| 冷凝液多      | 减少回流区、避免低压死角    |
